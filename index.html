Strut to spring migration - Chuyển đổi từ struts 1 sang spring

Đối với việc chuyển đổi từ struts sang spring ta sẽ có 4 việc phải làm:

  1. Chuyển đổi tất cả struts tags sang 1 thư viện khác ví dụ như theamlef, spring ( chỉ hỗ trợ một số tag) … Ta vẫn sẽ sử dụng lại thư viện jstl nếu muốn.

  2. Chuyển đổi tất cả action sang controller, việc này sẽ bao gồm cả xử lý thêm annotation để validate request. Việc trả giá trị ra màn hình sẽ phụ thuộc bạn sử dụng thư viện nào ở bước 1.

  3. Viết lại các class validate từ các file validate xml của project cũ nếu có.
    Custom annotation:

    Validator class:

  4. Chỉnh lại config.

  • Các file cấu hình của struts, file tag của struts, file validate.xml đều có thể xóa bỏ.
  • Có 2 trường hợp ở đây:
    • Nếu bạn muốn sử dụng spring boot và sau đó build một file chạy thì cần thiết cấu trúc lại thư mục cho đúng chuẩn của spring.
    • Nếu bạn vẫn muốn chạy trên server tomcat, thì không cần cấu trúc lại thư mục ( vẫn giữ nguyên thư mục webcontent ).
  • Cần sửa file web.xml, hầu như xóa hết để thay thể cấu hình spring sevlet ( việc cấu hình này bạn có thể sử dụng class ). Thêm file mvc-dispatcher-servlet.xml
